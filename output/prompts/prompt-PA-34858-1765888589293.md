# Overall Analytics Product Rules

## Overview
Overall Analytics provides a comprehensive view of user behavior and performance metrics across all products in the Valorem platform.

## Key Features
1. **Multi-Product Dashboard**: Aggregates data from all Insider products
2. **Performance Metrics**: Sessions, users, conversion rates, revenue
3. **Product Cards**: Individual sections for each product (Architect, Eureka, Email, etc.)
4. **Date Range Filtering**: Custom date selection with comparison mode
5. **Export Functionality**: Excel export of analytics data

## Testing Requirements

### Core Functionality
- Page load and initial data rendering
- Data accuracy and calculations
- Product card visibility and interactions
- Filter operations (date, metrics, goals)
- Export functionality

### Product Cards to Validate
- Architect (Journey Analytics)
- Eureka (Site Search)
- Email Campaigns
- Mobile App Analytics
- On-Site Campaigns
- WebPush Notifications
- Smart Recommender

### Data Validation
- Metric calculations (sessions, users, conversion rate)
- Comparison mode calculations
- Goal tracking and attribution
- Revenue tracking

### UI/UX Requirements
- Responsive design
- Loading states
- Error handling
- Empty states
- Tooltip information

### Performance
- Page load time < 3 seconds
- Data refresh without full page reload
- Smooth scrolling and interactions

## Common Issues to Test
1. Data not loading or infinite loading
2. Incorrect metric calculations
3. Product cards not displaying
4. Date range picker issues
5. Export failures
6. Filter not applying correctly
7. Layout breaking on different screen sizes
